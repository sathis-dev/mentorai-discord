## 15. FAQ CONTENT

### General Questions

**Q: Is MentorAI free?**
A: Yes! MentorAI is completely free with no premium tiers. All features are available to everyone.

**Q: How do I add MentorAI to my server?**
A: Use the `/invite` command or click the "Add to Server" button on our website.

**Q: What programming languages can I learn?**
A: MentorAI can teach ANY programming topic. Python, JavaScript, React, Java, C++, SQL, and 100+ more.

**Q: How does the AI work?**
A: MentorAI uses GPT-4 to generate personalized lessons, quizzes, and explanations on any topic.

### Feature Questions

**Q: How do I earn XP?**
A: Complete quizzes, lessons, claim daily bonuses, win challenges, and maintain streaks.

**Q: What are achievements?**
A: Achievements are badges you unlock for milestones like completing 10 lessons or maintaining a 7-day streak.

**Q: Can I compete with friends?**
A: Yes! Use /challenge for 1v1 battles or /arena for multiplayer competitions.

**Q: How does the streak system work?**
A: Use /daily every day to maintain your streak. Longer streaks give bigger XP bonuses.

### Technical Questions

**Q: Can I run code in MentorAI?**
A: Yes! Use /run to execute code in 15+ languages including Python, JavaScript, Java, C++, and more.

**Q: Is code execution safe?**
A: Absolutely. All code runs in sandboxed containers with no access to real systems.

**Q: Does MentorAI remember my progress?**
A: Yes! All progress is saved to our database and synced across all servers.
