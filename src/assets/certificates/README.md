# Certificate Assets Directory

This directory should contain background images and seals for certificate generation.

## Optional Assets

### Background Images
- `certificate_bg.jpg` - Default certificate background (1600x1200)
- `achievement_bg.jpg` - Achievement certificate background
- `mastery_bg.png` - Mastery certificate background

### Seal Images
- `mentorai_seal.png` - Main MentorAI seal (100x100 recommended)
- `achievement_seal.png` - Achievement seal
- `mastery_seal.png` - Mastery seal

## Note

If assets are not present, the certificate generator will use:
- Gradient backgrounds with decorative borders
- Text-based seals with the MentorAI logo

## Creating Custom Assets

### Background Guidelines
- Size: 1600x1200 pixels
- Format: JPG or PNG
- Style: Professional, elegant, subtle patterns

### Seal Guidelines
- Size: 100x100 to 200x200 pixels
- Format: PNG with transparency
- Style: Circular, badge-like design
