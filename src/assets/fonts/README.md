# Asset Fonts Directory

This directory should contain custom fonts for the card and certificate generators.

## Required Fonts (Optional - will use fallback if not present)

- `Roboto-Bold.ttf` - For headings and emphasis
- `Roboto-Regular.ttf` - For body text
- `Poppins-Bold.ttf` - Alternative heading font
- `Poppins-Regular.ttf` - Alternative body font

## How to Add Fonts

1. Download fonts from Google Fonts:
   - Roboto: https://fonts.google.com/specimen/Roboto
   - Poppins: https://fonts.google.com/specimen/Poppins

2. Place the .ttf files in this directory

3. The generators will automatically detect and use them

## Note

If fonts are not present, the system will fall back to system fonts (Arial, sans-serif).
